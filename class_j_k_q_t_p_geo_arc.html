Detailed Description

This JKQTPGeometricPlotElement is used to draw an arc.

A pie on linear and log-log axes (start angle: 10deg and end-angle 110deg):

Note
This class support the JKQTPGeometricPlotElement::DrawMode::DrawAsMathematicalCurve only. This means that arcs are always treated as mathematical curves, as no meaningful parametrization for arcs on non-linear axes could be found!
See also
Example (JKQTPlotter): Plotting Geometric Objects, JKQTPGeoChord, JKQTPGeoPie, JKQTPGeoEllipse
